### Option 1: Direct Install from Web
1. **Upload files to a web server:**
   - Upload `educational-room-game-enhanced.html` (rename to `index.html`)
   - Upload all `images/` folder files
   - Upload `sw.js` (service worker)

2. **Open in Chrome on Android:**
   - Visit your website URL
   - Tap the menu (‚ãÆ) ‚Üí "Install app" or "Add to Home screen"

### Option 2: Use GitHub Pages (FREE Hosting)
1. Create a GitHub account at github.com
2. Create a new repository called "medical-game"
3. Upload these files:
   - `index.html` (the game HTML)
   - `sw.js`
   - `images/` folder with all images
4. Go to Settings ‚Üí Pages ‚Üí Enable GitHub Pages
5. Visit the URL: `https://yourusername.github.io/medical-game`
6. On your Android phone, open the URL in Chrome
7. Tap menu ‚Üí "Add to Home screen"

### Option 3: Local File (No Installation)
1. Transfer `educational-room-game-enhanced.html` to your phone
2. Transfer the `images/` folder to the same location
3. Open the HTML file in Chrome browser
4. Bookmark it for easy access

## üÜò Troubleshooting

**Can't install?**
- Make sure you're using Chrome (not Firefox/Safari)
- Check that files are on a web server (not file://)
- Try using GitHub Pages (easiest method)

**Not working offline?**
- Make sure `sw.js` file is uploaded
- Reload the page once while online

**Images not showing?**
- Check that `images/` folder is in the same directory
- Image files must be named exactly as referenced
